# UC-01: Validate JSON Structure and Schema Compliance

**Use Case**: Verify that JSON documents are syntactically correct and comply with their schemas.

## Prerequisites

- Python 3.11+ installed
- Repository cloned locally
- Dependencies installed: `pip install -r requirements.txt`

## Steps

### 1. Navigate to Repository

```bash
cd /path/to/MWDR-Showcase
```

### 2. Validate All Documents

```bash
python3 src/validate/validate_json.py --path .
```

**Expected Output**:
```
Validation Summary
==================
total_files_scanned: 5
parsed_ok: 5
syntax_errors: 0
schema_valid: 5
schema_invalid: 0
schema_missing: 0

Result: PASS
```

### 3. Generate Detailed Report

```bash
python3 src/validate/validate_json.py --path . --json-report validate_report.json
```

This creates `validate_report.json` with:
- Metrics (counts for each category)
- Lists of errors (if any)
- Schema routing information

### 4. Validate Specific Module

```bash
python3 src/validate/validate_json.py --path modules/demo_module/
```

### 5. Validate Single File (if supported)

```bash
python3 src/validate/validate_json.py --path modules/demo_module/documents/MAP-WORLD-PTOLEMY-0150.json
```

## Troubleshooting

### Syntax Errors

**Error**: `syntax_errors: 1`

**Solution**: Check JSON syntax using:
```bash
python3 -m json.tool path/to/file.json
```

Common issues:
- Missing commas
- Trailing commas
- Unquoted keys
- Single quotes instead of double quotes

### Schema Validation Errors

**Error**: `schema_invalid: 1`

**Solution**:
1. Check the validation report for specific error message
2. Compare against schema: `shared/schemas/epistemic_documents/epistemic_document.schema.json`
3. Verify required fields are present
4. Check data types match schema requirements

### Schema Not Found

**Error**: `schema_missing: 1`

**Solution**:
1. Check `src/validate/schema_routing.json` for pattern matching
2. Verify schema file exists at specified path
3. Add routing rule if needed

## Expected Results

**Successful validation**:
- `syntax_errors: 0`
- `schema_invalid: 0`
- `Result: PASS`

**Informational**:
- `schema_missing: N` may be non-zero (files without schemas are informational)
